Law Talk Chatbot

The Legal Query Response System is a web-based application designed to provide users with relevant legal information based on their queries. Built using Python, Flask, and MongoDB, this system combines user-friendly features with advanced natural language processing techniques to deliver accurate and relevant legal responses. The application utilizes a pre-existing dataset containing legal queries, responses, applicable legal articles, and usage information. By implementing the TF-IDF (Term Frequency-Inverse Document Frequency) model and cosine similarity, the system efficiently matches user queries with the most suitable responses. Users can register and log in to the system, ensuring that their sessions are secure and personalized. The registration process involves securely hashing passwords using the werkzeug library, and the login system checks these hashed passwords for authentication. Once logged in, users can submit their legal queries through the interface, and the system returns the most relevant response along with details of the applicable legal articles. The application also keeps a history of user queries, allowing users to revisit previous searches easily.
